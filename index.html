Thank you for downloading MinGW!

Follow these steps to install mingw on your device!

1. Make sure mingw.zip file has been downloaded. If not, you can download it by clicking here .

2. Unzip mingw.zip file.

3. Step into extracted mingw directory and run installer.exe.

4. Use MinGW installation manager to configure the installation.

Congratulations! You succesfully installed MinGW.
